# geodecoder

Convert from format:

```
<name>:
<type of checking>:
<address>
<plan url>
<address>
<plan url>
<type of checking>:
<address>
<plan url>
<address>
<plan url>

<name>:
<type of checking>:
<address>
<plan url>
<address>
<plan url>
<type of checking>:
<address>
<plan url>
<address>
<plan url>
```

...to GeoJSON that can be accepted by Yandex's [map constructor](https://yandex.ru/map-constructor/).
